# Codex Toolkit

This directory contains the first-cut Codex version of the Claude toolkit.

Files:

- `AGENTS.md` defines review and helper roles for Codex.
- `AGENTS.template.md` is the project-entry template to copy to a future repo root as `AGENTS.md`.
- `RULES.md` defines default repository behavior and coding standards.
- `WORKFLOWS.md` defines recurring task procedures such as running the pipeline, compiling the paper, and reviewing files.
- `SETUP.md` explains how these files play the role that `CLAUDE.md` plays in Claude Code.

Current status:

- project rules are defined
- recurring workflows are defined
- setup guidance is defined
